Práctica 1: Cola de impresión en C

Luis Ángel Martínez Zamaniego

1. Introducción

En este proyecto se desarrolló un sistema de simulación de una cola de impresión utilizando el lenguaje C. Se modela el comportamiento de una impresora que procesa trabajos en orden FIFO.

2. Diseño

Se utilizó una estructura tipo cola mediante listas enlazadas dinámicas para permitir flexibilidad. 


typedef struct Trabajo {
    int id;
    char usuario[32];
    char documento[32];
    int total_pgs;
} Trabajo;
